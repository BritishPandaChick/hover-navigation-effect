# Smooth Hover Effect
Web page using a smooth hover Effect to reveal web page navigation

## Tech Stack
- HTML
- CSS

## Instructions
1. Clone repo or download zip file. 
2. Open index.html file in your browser.
3. A message saying "Hover anywhere to reveal navigation" will appear in the center of the white page. When you hover in any spot, the animation will push the white page back to reveal the navigation menu on the right side of the page with a red background.
